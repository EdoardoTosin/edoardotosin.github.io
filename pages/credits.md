<br>

This website is powered by [Jekyll SSG](https://jekyllrb.com/) and hosted using [GitHub Pages](https://pages.github.com/). The theme is based on [Jekyll Garden Theme v0.4](https://github.com/Jekyll-Garden/jekyll-garden.github.io/tree/87e7c91d4ef3280271920844fad2769bd093bf04) with custom modifications.

## Core Technologies

- **[Jekyll](https://jekyllrb.com/)** - Static site generator
- **[GitHub Pages](https://pages.github.com/)** - Hosting platform
- **[Bulma](https://bulma.io/)** - CSS framework for responsive layouts
- **[Liquid](https://shopify.github.io/liquid/)** - Templating language

## Styling & Design

- **[IBM Plex Sans](https://fonts.google.com/specimen/IBM+Plex+Sans)** - Primary font for text
- **[Cascadia Code](https://github.com/microsoft/cascadia-code)** - Monospace font for code blocks
- **[Fruity Pygments Theme](https://github.com/jwarby/jekyll-pygments-themes)** - Syntax highlighting
- **[Feather Icons](https://feathericons.com/)** - Icon set

## Image Gallery & Lightbox

- **[PhotoSwipe v5](https://photoswipe.com/)** - Touch-friendly lightbox gallery
  - Custom carousel component with touch/swipe support

## JavaScript Libraries

- **[KaTeX](https://katex.org/)** - Fast math typesetting
- **[Mermaid](https://mermaid.js.org/)** - Diagram and flowchart generation
- **[Fuse.js](https://www.fusejs.io/)** - Lightweight fuzzy-search library

## Content Management

- **[Obsidian](https://obsidian.md/)** - Knowledge management and note-taking
- **Markdown** - Content writing format

## Development Tools

- **[Sass/SCSS](https://sass-lang.com/)** - CSS preprocessing
- **Git** - Version control
- **Ruby & Bundler** - Dependency management

## Features & Enhancements

- Responsive design
- Dark/light theme support with system preference detection
- Print-friendly styling
- Sidenotes and margin notes
- Internal wiki-style linking
- Tag-based content organization
- Search functionality
- RSS Atom feed generation

## Special Thanks

Thanks to the open-source community and all the developers who created and maintain these tools and libraries.
